No Wifi? No Problem!

编辑 由 Resume 2个星期前创建，共65个应用

为之后要开始的暑假两个月无WiFi生活作准备ε(┬┬﹏┬┬)3
如果没有办奇奇怪怪的卡的话（没错说的就是王卡和2233卡），没WIFI时就只能看看小说，包包定向流量包听歌，打打游戏了吧(＞﹏＜)，还可以听听有声书什么的
应用集中所有游戏均能离线单机玩耍！
应用集名抄袭了Google Play的同名栏目，不过那个栏目里面好多游戏我都不很喜欢。。

  • Batman

    可能我真的是假的蝙蝠粉，老爷有这么多套平行世界装扮我竟然不知道。 LV30就封顶了，也不是开放世界四处走动的游戏，可能有粉丝要失望了。但是其实建模的精细程度和动作设计都是挺棒的。 大概900M，算是大游戏吧，我用GP Game ID登录的，游戏进度存储在WB云端（如果有WB账号的话还能送两套装扮）

  • Injustice 2  来自Google Play

    最近才出的不义联盟2，首先角色比第一代好看了，招式也有了重新设计，战斗的灵活度增大了。还是3v3的设计，还是角色坑钱的套路。 890M 还带了剧情，像看大片一样好带感

  • GOF2

    在银河游荡，采矿打海盗，目标是回家（听起来很套路的剧情） 场景是自由的，星系为背景，飞来飞去很有感觉，唯一感觉不正常的地方是操作杆往上扳的时候飞机是往下的(o-ωｑ)，然而左右又是没问题的。玩了半个小时连个操作杆都没有练熟气屎了（我这个手残估计得卸掉(￣┰￣*)，难得玩个游戏开心一下还要受这气） 里面的角色头像风格都很奇葩ヾ(´∀`o)+，可以拿出来做暴走表情包 这游戏没有云同步，跟小时候玩的那些老式电脑大游戏一样是读本地存档的

  • 反击间谍

    这游戏有些年头了，最近一次更新是在14年8月，Play售价4.99美元 游戏的背景很带感，你是一个暗中执行任务的间谍，要偷取核弹计划图阻止苏联成功发射核弹，所以如果你失败了可以认为对你国人就是世界末日了。 画面很有特色，毕竟是PlayStation的游戏。音乐的紧张程度根据游戏的不同会发生变化。武器多样。最开始就是主角和敌人正面交锋的动画和音效吸引了我

  • 超凡蜘蛛侠

    说起这个游戏我身上就涌出阵阵回忆之流╰(￣▽￣)╭ 市场售价6.99美金，但是我在两年前玩的是破解版，是我玩过的第一款三维开放世界大型游戏，穿梭在建筑物中营救伤员帮助警察，和恶人作战，各种特技DuamgDuangDuang 续作超凡蜘蛛侠2也是6.99美金，画面更漂亮了，任务难度也增大了一些，如果只想玩一款的话直接玩第二部就可以了。

  • 侠盗猎车手之圣安地列斯:Grand Theft Auto

    R星的游戏，6.99美元 也是主线剧情+开放沙盒游戏，游戏也有些年头了，所以里面的动画建模什么的可能会显得没有现在的一些新游戏真实，而且经常出现相机图像的可见性条件没有设置好结果相机前方出现较多环境面片遮挡主场景的情况(拒绝相信这是我手机的问题，因为玩其他大型游戏都没这种问题)(✽ ﾟдﾟ ✽)

  • arrow.io

    打开手机，移动你的小手指，射射射！✧(≖ ◡ ≖✿) 小型联网离线均可玩的游戏 支持5V5ʕ•ٹ•ʔ

  • BIA3

    Gameloft的一款射击游戏，通过一位老兵的回忆展开，蹲堡垒，扔手榴，但主要还是靠手枪♂ 这家大厂你们懂的，质量高，老是做很大型的游戏，从老少皆宜的迪士尼建设类游戏到十七禁的赌城风云 不过到现在不知道游戏进度怎么云同步。。感觉大厂不做云同步不正常啊是不是我不会用(*￣︿￣)

  • Gangstar 4

    楼上提到的十七禁游戏之一，赌城里面闹了事还想活下去的一个大块头在黑暗势力翻涌的地点低调生活。 开放沙盒类，不玩主线剧情的时候就自己瞎转悠，抢了几辆车撞了好多东西（和人以后）被警车追捕，然后我想了想就停车，等那个警察出来就把警车开走了（〃｀ 3′〃）

  • Zombie

    预警：一定要联网才能玩 看截图我还以为是动作型团战就下下来了，结果原来是策略类。但是画风我很喜欢就先留着了。 今天还下载了个Time Shift也是一定要联网的，不过射击类游戏我已经有不用联网的BIA了，所以就把它卸了

  • Cut the Rope Time Travel

    割绳子这么有名就不多介绍了。 很久以前玩通关了，最近打开发现又更新了很多关卡，是时候再次踏上征途了(*￣ω￣)

  • Fallout Shelter

    养成游戏，给后核战时代的人们营造一个舒适的地下生活环境，饮食用水电力科技样样都要管。 好了情景你们都知道了但是看你们喜不喜欢还是要自己下下来体验一下画风和音效Ψ(￣∀￣)Ψ 我并不喜欢养成游戏，但是这个游戏在我手机上存活下来了，不知道刺中了我哪个G点(❤´艸｀❤) GP Game 云同步

  • 我的世界 Minecraft

    这么多人问怎么没有MC。。还是加进来好了，毕竟是一代神游 可是我没有玩过呀，怎么给你们写点评呢(｡ŏ﹏ŏ) 小编的点评也辣么短，抄袭不能( •̀ㅁ•́;) 咳咳…这是一款可以离线玩耍的沙盒类神游，打造属于你的世界，特色就是可以让我的室友呆在床上不吃不喝不起床(这也正是我不敢玩这个游戏的原因)

  • Frozen Free Fall

    三消游戏我最喜欢这个，迪士尼也是懂人心，冰块碎掉的声音和动画效果简直舒爽，再加上一些魔法的元素，美滋滋。 GP Game 云同步

  • Google Play Games

    玩游戏刷成就，以前老是用什么360啊腾讯游戏啊什么的简直黑历史。也不是说他们搞的游戏不好，关键是种类不够多，比谷歌的少多了，少了很多美

  • Google Play Store

    没这个很多游戏和应用打不开的 现在我一般是逛菜市场和酷安同时逛，但是发现的好多应用小编也是不上架的，有点小难受。

  • Hungry Shark

    饥饿鲨，最大的特色是可以吃人，，，呸呸呸，太重口了。最大的特色是开放世界到处探索，380M的游戏，地图应该是挺大的。 加速！深潜！旋转跳跃(≧∇≦) 登录Facebook即可云同步游戏进度

  • Plants Vs Zombies 2  来自Google Play

    PVZ2 玩的时候可能没有当年在电脑玩第一版的新鲜感，但是其实一如既往的幽默感，精心的配乐和。。。。好吧我编不下去了，反正好玩就是了 GP Game 云同步

  • PvZ Heroes

    PVZ卡牌游戏，可能是抄袭的clash royal(不知为何忘了那游戏的中文名了)。 由于是PVZ粉，我就玩了 GP Game 云同步

  • Pocket Mortys

    看过Rick and Morty的一定会很有亲切感，美剧脑洞大，各种恶搞和美式幽默，看完又能触动人心引人深思。 游戏秉承了对应美剧的恶搞风格，看这名字就知道是一款恶搞口袋妖怪的游戏。游戏配音也是跟演员一致的。。。喜欢口袋妖怪的可以玩玩尝鲜，原剧粉也应该好好体会一下官方手游

  • RoboCop

    安卓7.0 ZUI2.5，无限闪退(・へ・)，看来是与这个游戏无缘了。本来看游戏介绍和截图还热血沸腾地来着

  • Stampede

    我有一头小毛驴我从来也不骑，我最爱骑的还是那，村口长毛象嘿，跑过一匹又一匹 养成+休闲点击类游戏 GP Game 云同步

  • Temple Run 2

    老游戏了，跑酷游戏经典啊 玩跑酷就是看画面看场景，这游戏几个新主题都很精彩的样子 GP Game 云同步

  • 逃亡未来2

    休闲跑酷，音乐很棒，剧情的加入也让人玩得更投入，暗影风格很好看，但是人物动作的动画效果我觉得有点拖沓。。

  • 滑雪大冒险2

    相比于扁平幽静的Alto，我还是比较喜欢这个的拟物风格和滑稽音乐。 通过登录GP Game ID或者Facebook同步游戏进度 唯美风的Alto支持GP Game云同步

  • TripleTown

    另类三消，平静的音乐，平静的场景，平静的小熊熊 也是拟物风。 这种没有关卡没有等级的游戏不需要云同步，不过购买的物品可以通过GP Game ID 记录在云端

  • N.O.V.A.近地联盟先遣队：传承战记

    只有20M。 画面震撼的第一人称射击游戏，情节紧张，配乐到位 然而我有一次启动以后游戏进度全失，就不想再玩了 同厂的那个同样小巧的足球游戏只有联网才能保存编辑的球队到云端，一离线就变成离线账户了

  • 几何竞赛

    画风很有特点，简单的几何图形，鲜亮的配色，但是这种画风也很容易让人视觉疲劳，像我这样的手残就一直各种撞机。。。 GP Game云同步

  • 炫光弹球

    画面很炫的三维弹球游戏，勾起了童年的回忆。 不过这个开发者开发的游戏都是虽然没广告，但不买高级版就只能每次都从起点开始，不记录游戏存档点。

  • 真实拳击

    UFC老是提示连接不上EA服务器，所以就换了这个来玩。打击感没有UFC强，角色也没有UFC多都不能愉快地搞比利了(手动滑稽)

  • 模型移动Mekorama

    类纪念碑谷游戏，五十个关卡，游戏只有5M。能自己做关卡，可以导入别人做的关卡。一个字，强。 至于纪念碑谷是什么我想就不用解释了，开创一个游戏新时代的作品，隔一段时间我还会重温一下

  • 绝处逢生

    游戏还在Demo阶段，有纪念碑谷的感觉，但是又加入了光线扫略的创意，很期待它的正式版。

  • 忍者岚

    暗影风格忍者闯关 音乐紧张有节奏，动画流动感很不错 等第三个主题好久了就是不出(｡ŏ﹏ŏ)

  • Dokuro多克罗

    这游戏我玩了两三遍吧，粉笔画风格简直不能更赞 大致剧情是王子变小骷髅，骷髅爱上傻公主，带公主逃出魔堡，公主王子性福生活。童话故事一般的温馨剧情。 游戏是半动作半策略的闯关冒险，整个游戏有很多章节，走遍整个魔堡，打遍全部Boss，但是对于小骷髅来说只有三个阶段，骷髅身段为公主排除万难阶段，可以短时变身王子抱抱公主阶段，还有公主被抓走悲极生力量化身超级王子的阶段 市场售价1.99美金

  • 虚拟盆栽

    真实时间养成，画面音乐给满分。 可是真实时间为什么不直接去买个真的盆栽呢(－－〆)

  • 口袋植物

    这个画风戳到我萌点了，简直了 好像除了这个我就说不出什么特点了，毕竟我不玩养成的，因为我耐心不够用(✽ ﾟдﾟ ✽)

  • 舞动的线

    真虐心，没有什么比听着好听的音乐结果因为一个失误音乐戛然而止更让人痛心的了 (官方把OST发了专辑，好评😋)

  • 独裁者2：进化

    独裁者系列的游戏都是策略养成类的，通过卡牌游戏攻占新土地，而对已有的国土需要建筑一些东西，还要听取不同阵营的人的上奏，给出无法反悔也无法被推翻的独裁决定，但是这些决定影响了你的不同类型的人口，最后会影响到占领新土地作战时的卡牌选择上

  • OPUS地球计画

    剧情不知道赞不赞，因为这种探索剧情类我个人没有耐心。。。 我只能说画面挺赞，喜欢剧情游戏的玩家给的评价都挺高的

  • Implosion聚爆

    1.8GB，所以画面特效很赞这一点就不言而喻了。剧情和音乐很带感，打击感很强，炫光就像炫迈，转啊转啊停不下来 不买的话可以试玩第一章的好几关 GP Game云同步

  • 沉默年代

    沉默年代，或许不该，太遥远的相爱～～ 跑题了… 剧情解谜类游戏，让我找回了当初玩“Stanley博士的家”的感觉。 第一部免费，第二部要钱

  • 深海水族馆 AbyssRium

    点击类养成游戏 真的是点击类游戏。。点点点可以获得活力来升级火山珊瑚和产生小鱼 五光十色的水族馆哦 GP Game 云同步
